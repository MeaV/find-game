# Find Objects game for kids (1-3)
Simple game made in React & Redux for my kid and me :)

## Requirements
- node 6
- webpack & babel

## Install & run
    npm i webpack babel-cli -g
    npm i
    npm start

### build
    npm run build

### test
    npm test
